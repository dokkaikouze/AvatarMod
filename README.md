## AvatarModとは

VRChatのアバター向けの各種設定済みファイルの詰め合わせです  
主に以下のような内容になっています  
・処理負荷の軽量化設定データ  
・PhysBoneや表情、ハンドサインなどの設定データ  
・メニューから明るさ調整や衣装変更などの追加機能  
・その他バグ修正など

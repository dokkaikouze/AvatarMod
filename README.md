## このリポジトリについて

VRChatのアバター向けの各種設定済みファイルの詰め合わせです
主に以下のような内容になっています
・処理負荷の軽量化設定データ
・PhysBoneなどの設定データ
・メニューから明るさ調整などの追加機能
・その他バグ修正など
